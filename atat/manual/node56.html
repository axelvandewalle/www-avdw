3d2gnu 

3D to GNUplot 3.50, by Axel van de Walle
 -if=[string]   Input table file (Default: tri.bi)
 -tc=[string]   set terminal command (Default: set term postscript eps color ; set out 'test.eps')
-eye=[real],... Eye position (Default: 0,-5,0)
 -at=[real],... Lookat position (Default: 0,0,0)
 -up=[real],... Up direction (Default: 0,0,1)
-fov=[real]     Field of View (in degrees) (Default: 30)
-lit=[real],... Light direction (Default: 0,0.707107,-0.707107)
-amb=[real]     Ambiant light level (Default: 0.5)
  -h            Display more help (Default: Off)
  -d            Use all default values (Default: Off)
