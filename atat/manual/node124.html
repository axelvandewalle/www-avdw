str2poscar 

Converts an ATAT structure file (from stdin) to the VASP POSCAR format (stdout).
Options:
  -old  output old-style POSCAR (no atom label).
  -r    do the reverse conversion (VASP -> ATAT).
         works for both old- and new-style POSCAR.
