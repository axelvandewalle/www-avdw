<H1>Alloy Theoretic Automated Toolkit (ATAT) Home Page</H1>

<IMG SRC="atat.gif" ALT="[ATAT logo]" ALIGN=middle><P>

ATAT is a generic name that refers to a collection of alloy theory tools
developped by <A HREF="http://alum.mit.edu/www/avdw">Axel van de Walle</A>, in collaboration with various <A HREF="SUB:collaborators">research groups</A> and
with various sources of <A HREF="SUB:finan">financial support</A>.<P>
